# 🧠 VitaTrack - Daily Lifestyle Health Risk Predictor

A full-stack health monitoring system that analyzes lifestyle habits and predicts health risk using intelligent scoring and visualization.

---

## 🚀 Features

- ✅ Health Score Calculation
- ⚠️ Risk Level Prediction (Low / Medium / High)
- 🤖 AI-based Lifestyle Analysis
- 📊 Radar Chart Visualization
- 📈 History Tracking Chart
- 💾 Data stored in MySQL Database
- 🔗 Full Stack Integration (Frontend + Backend)

---

## 🛠 Tech Stack

| Layer      | Technology |
|-----------|-----------|
| Frontend  | HTML, CSS, JavaScript |
| Backend   | Spring Boot (Java) |
| Database  | MySQL |
| Charts    | Chart.js |

---

## ⚙️ How to Run

1️⃣ Run Spring Boot backend  
→ Port: `http://localhost:8081`

2️⃣ Open frontend  
→ Open `index.html`

3️⃣ Enter health data:
- Sleep
- Exercise
- Water Intake
- Junk Food
- Heart Rate
- Stress

4️⃣ View:
- Health Score
- Risk Level
- Charts & Predictions

---

## 🧠 How It Works

- User enters lifestyle data
- Frontend sends data using `fetch()`
- Spring Boot processes and stores data
- Risk level is calculated
- Charts visualize results

---

## 📊 Output

- Health Score (0–100)
- Risk Level Indicator
- AI Prediction Score
- Lifestyle Radar Chart
- History Graph
